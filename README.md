Fishy — приложение с открытым исходным кодом для учёта и планирования отгрузок, задуманное как цифровая замена блокнота для сюрвейеров, призванное автоматизировать контроль процесса — от планирования отгрузок до составления отчёта, минимизируя рутинные проверки и случайные ошибки.

Основные возможности:

• Проверка номеров контейнеров и вагонов по контрольной цифре согласно стандарту ISO 6346.

• Система двойного контроля для разделения счёта вывезенных и завезённых поддонов.

• Сохранение завершённых отгрузок в архиве с возможностью экспорта отчёта.

• Планирование отгрузок с возможностью импорта введённых данных.

• Создание чек-листов для каждой запланированной отгрузки.

• Информативные уведомления об отгрузках.

• Автоматическое и ручное сохранение черновиков.

• Сохранение шаблонов часто используемых данных контрагентов, продукции и другой служебной информации.

Fishy не требует подключения к интернету, все ваши данные хранятся только на вашем устройстве и абсолютно конфиденциальны.
